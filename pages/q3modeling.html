Quake III Arena Player Model Creation in 2024
Tools:

  1. Any Blender version that exports to gltf (.glb) | For Modeling, UV, Animation work
  2. Noesis | To convert gltf/glb to md3
  3. Npherno's MD3 Compiler | To convert triangles to tags
  4. Maverick Model 3D | To split up animation and make animation.cfg and composite the model
1. Model, animate and texture in blender 4 with restrictions:
  • Use template and provided orientation!
  • Budget: 999 tris per surface->(separate blender object)
  • Name the Mesh data (as h_head, l_lower, u_upper) not the Object!
  • Material and texture name should be the same!
  • Make triangles for tags!


2. Export as .glb - Animation: NLA Tracks; Orientation: No +Y up!
3. Import the .glb to Noesis and then export it as is to .md3, no extra settings required.
4. Import the .md3 to Npherno's MD3 Compiler to turn the triangles into tags and export.
5. Import the .md3 to Maverick Model 3D and split the animations and export. Follow specifications.
